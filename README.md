# IPL Match Analysis & Prediction

## 📌 Objective
Analyze IPL match data and build a machine learning model to predict match outcomes.

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn / Matplotlib
- Scikit-learn

## 📊 Data Analysis
- Identified top-performing teams based on match wins
- Analyzed impact of toss on match outcomes
- Studied venue-based performance trends

## 🤖 Machine Learning Model
- Used Logistic Regression to predict match winners
- Applied One-Hot Encoding for categorical features
- Features used:
  - Team1
  - Team2
  - Toss Winner
  - Venue
  - Toss Decision

## 📈 Results
- Achieved moderate accuracy (~50–65%)
- Demonstrated how feature engineering improves model performance

## 💡 Key Insights
- Certain teams consistently outperform others
- Toss and venue have a noticeable impact on match results

## 🚀 Conclusion
This project demonstrates how data analysis and machine learning can be combined to extract insights and make predictions in sports analytics.
